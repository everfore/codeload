# 	codeload

github codeload url(sample): https://codeload.github.com/everfore/codeload/zip/master

##	get

go get github.com/everfore/codeload

##	usage

>codeload

>user/repo:master

	or

>codeload

>user/repo

if branch is nil, use master as default branch.
